##<a id="md_2" name="md_2"></a>准备工作##


安装并配置Python,Python[下载地址](https://www.python.org/downloads/)


安装并配置Node.js,Node.js[下载地址](https://nodejs.org/en/download/)

安装Express.js


>$ npm install express -g


安装bower

>$ npm install bower -g


安装程序依赖包

>$ cd learn_html5

>$ npm install

>$ bower install

运行示例程序

>$ npm start

在浏览器输入：
>`http://localhost:3000/`
